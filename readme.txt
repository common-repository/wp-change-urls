== Installation ==

1. Upload `wp-change-url.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Change settings in admin panel

== Frequently Asked Questions ==

Q: How exclude any url? 
R: Enter to /wp-admin/options-general.php?page=wp-change-url.php, copy from text frame "New" to text frame Exclude.
